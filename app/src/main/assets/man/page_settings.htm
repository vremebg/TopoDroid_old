TopoDroid settings

GENERAL SETTINGS
  • Work directory. Default "unset" (ie "TopoDroid")
  • Text size: size of the text in lists. >= 1. Default 14
  • Buttons size {small, normal, medium, large, huge}. Default normal
  • Level of features {basic, normal, advanced, expert}. Default normal
  • Keyboard. Enable custom keyboards instead of Andrid default keyboard.
  • Cosurveying service. Default OFF
  • Language {default,en,fr,es,it,ru,de,pg,slo,bg,cn,ukr,hu}. Default "default" (i.e. locale)

IMPORT/EXPORT SETTINGS

  • PocketTopo color map Map PocketTopo colours to TopoDroid objecs
  • Estimate LR extend on import of Compass and Visual Topo files, Default YES
  • Default data export type {th,svx,dat,srv,top,tro,dxf,svg,csv} default th
  • Default scetch export type {none,th,csx,dxf,svg,png} default th
  • Therion maps first: export therion maps before survey data. default OFF
  • Therion stations: Add station points to Therion scraps. Default YES
  • Therion splays: Add splay lines to Therion scraps. Default YES
  • Compass station prefix. Default NO
  • Compass LRUD splay angle. LRUD minimum splay-leg angle.
  • Compass swap LR. Swap L and R on Compass export.
  • Survex/Topo line termination {Linux,Windows}. Default Linux
  • Survex splays TO station. Default NO
  • Survex LRUD. Add LRUD to Survex export. Default NO
  • SVG grid. Add grid to SVG export. Default OFF
  • SVG in HTML. Export SVG as HTML page. Default OFF
  • KML stations. Add stations to KML export Default ON
  • KML splays. Add splays to KML export Default OFF
  • PNG resolution [0.5, 10]. Default 1.5 [x200 px]
  • PNG background color (RGB in 0-100). Default "0 0 0" [black]
  • DXF sketch scale. Default 1
  • DXF splay endpoints. Default ON

SURVEY DATA

  • Team. Default survey team. Default "unset"
  • Stations Policy: Survey stations naming policy {none, splay+fore, splay+back, fore+splay, back+splay, back+splay+fore, tripod}. Default "splay+fore".
  • Station names {alphanumeric,numeric}. Default alphanumeric
  • Initial station name. Default "unset"
  • Photo thumbnails size [80,400]. Default 200 pxl
UNITS
  • Length units {m,ft}. Default m
  • Angle units {deg,grad}. Default deg
  • Sketch grid units {m,yd,2ft}. Default m
SHOT DATA
  • Leg tolerance. Tolerance among the shots of same leg > 0 [percent]. Default 0.05 (5%)
  • Minimum Leg shots. Minimum number of shots per leg >=2. Default 2
  • Splay vertical threshold. Limit for splay extend "vertical" [0,90]. Default 10 deg
  • Leg vertical threshold. Leg V threshold [0,90]. Default 80 deg
  • Settable "extend" reference. Settable or fixed extend L/R reference. Default NO
  • Loop closure. Default "none"
  • Prev/Next buttons. Default ON
  • Backsight in user entered shot dialog. Default NO
  • Magnetic anomaly compensation. Default NO
  • Shot direction delay (timer). Default 10 s
  • Shot direction timer beep volume [10,100]. Default 50 percent
LOCATION
  • Location units {ddmmss, dec.deg}. Default ddmmss
  • CRS. Name of CRS like in Proj4. Default Long-Lat
ACCURACY
  • Acceleration tolerance > 0. Default 400
  • Magnetic field tolerance > 0. Default 300
  • Magnetic Dip tolerance > 0. Default 3 deg

SKETCHING

  • Drawing tool picker {recent,list,grid,triple grid}. Default recent
  • Recent tool number {3, 4, 5, 6}. Default 4
  • Side drag. Enable side dreg in sketch window. Default OFF
  • Zoom controls {off,temporary,permanent}. Default permanent
  • Midline check. Check if stations are attached to midline. Default ON
  • Backup number. Default 5
  • Backup interval. Default 60
POINT TOOLS
  • Non-zoomed point icons. Default NO
  • Point tools scale. Default 1.2
  • Labels size. Default 24 px
LINE TOOLS
  • Lines width (N.B. wall line are twice this). Default 1 px
  • Line style {Bezier,fine,normal,coarse}. Default normal
  • Line-point spacing. Minimum distance between line points. Default 10 px
  • Section lines "direction tick" size [1,20]. Default 8 [x unit]
  • Section points. Add Therion section point to section line. Default ON
  • Area border visibility. Default YES
  • Bezier line accuracy. Default 1
  • Bezier line corner threshold. Default 20
SKETCH ITEMS
  • Survey line width [0.5,10]. Default 1 px
  • Station name size. Default 24 px
  • Green dots radius [0.5,100]. Default 5 px
  • Selection radius >= 0. Default 24
  • Eraser size. Default 36
  • Shift sensitivity. Default 60
  • Pointing radius. Default 16
  • Splay plan-view threshold [0,90]. Default 80 deg
  • Splay dashed clino threshold [0,90]. Default 50 deg
  • Splay dashed angle threshold for x-sections [0,90]. Default 60 deg
  • X-section H threshold [0,90]. Default 70 deg
AUTO-WALLS
  • Walls type {none,convex}. Default none
  • Splay plan clino. Default 70 deg
  • Splay profile clino. Default 45 deg
  • Separation. Default 0.1 m
  • Concavity. Default 0.1
  • Step. Default 1 m

DEVICES

  • Bluetooth. Check if BT is enabled on start. Default YES
  • Connection data mode {on-demand,continuous}. Default on-demand
  • Auto reconnect (in continuous mode). Default FALSE
  • BT socket type {normal,insecure}. Default normal
  • Z6 workaround. Default YES
  • Connection delay >= 0. Default 2
  • Autopairing. Default YES
  • Data pause [ms]. Default 50
  • Data ready wait delay [ms]. Default 100
  • Laser pause [ms]. Default 1000
  • Shot pause [ms]. Default 4000

CALIBRATION SETTINGS

  • Group policy {4x4+M,Nx4,distance). Default Nx4
  • Group tolerance. Default 40 [deg]
  • Calibration epsilon. Algorithm error. Default 0.000001
  • Calibration maximum iterations. Default 200
  • Show calibration raw data. Default NO
  • Calibration algorithm {auto,linear,non-linear}. Default auto
