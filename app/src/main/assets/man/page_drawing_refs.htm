Sketch references

This dialog is opened by the Reference button of the Sketch window.

  • legs, survey midline (white)
  • splays (grey)
  • stations names (violet)
  • grid, metric or yard
  • scalebar, reference scalebar and North/Upward arrow
  • outline, outline of another sketch
There are three grids,
  1. a grid at 1 unit spacing (dark grey)
  2. a grid at 10 units spacing (grey)
  3. a grid at 100 units spacing (light grey)

The grid units can be 1m, 1y, 2ft, 0.1m.
TopoDroid shows only the grids that are appropriate for the scale of the sketch display.

The scalebar is adapted to the grid units (m, y, ft, dm) and has a small arrow indicating the North/Upward direction.

If there are other sketches, the outline of another sketch can be displayed (in grey) to help drawing the sketch. Checking the box opens a new dialog to pick the sketch to display, or to clear the outline display.
The display of the outline of the x-sections can be turned on and off thru the edit dialog of the section points.
The display of the outlines is cleared when the sketch is closed.

A further checkbox [E] specifies whether the reference should be held fixed when shifting the drawing with two fingers. This is used to adjust a sketch imported from PocketTopo or Therion to the reference in the case it gets imported with a translation. While you touch the screen with a third finger the zoom is applyed only to the sketch.
