[9] SKETCH WINDOW

The Sketch button of the "Shot data" window opens the dialog with the list of sketches, besides the "New Sketch" button. Tap on this button to create a new survey sketch.

Each sketch is made of two sketch files: one for the plan view, the other for the profile view (either extended or projected). X-sections have only one file. A new sketch is opened in the plan view. Afterwards the sketch dialog shows two columns, one for the plan, the other for the profile view, and you can choose which one to open from.

The extended profile view displays the midline with the shots drawn left or right, or vertical, according to their "extend". Splays are projected on the vertical plane of the shot that is closest in azimuth, unless the user has set a given "extend" through the shot edit dialog. For example, for the plan view in figure, the splays in the light region are projected on the vertical plane of shot 1-2, those in the dark region on the vertical plane of 0-1.

The projected profile view displays the midline and the splays projected in the vertical plane perpendicular to the projection azimuth.


Splays
Splays are displayed with continuous or dashed/dotted line and with different colors.
The splay display is turned off if the shot is marked for no-display in plan or profile view.
A "commented" splay is dark gray. X-splays are green. Otherwise the splay is gray.

In plan view if the inclination of the splay is over the plan-view threshold, the splay is not drawn. If the splay clino is above the dash clino-threshold, it is dotted (positive clino) or dashed (negative clino).

In profile view the splay is dashed/dotted if the angle with the "closest" leg in azimuth is above the dash azimuth-threshold. However is the splay dash-coherence is checked the splays are dash/dotted as in plan view.

For x-sections the x-splays are green, the others are blue (or light blue), and dashed (or dotted) if the splay angle with the section plane is over the dash angle-threshold.

The sketch window has four modes:

  • move, the default: you can shift the canvas, and zoom in/out. If the "fixed references" box in the references dialog is set, shifts affect only the drawings [level experimental]
  • draw: in this mode you add iconic symbols (points, including labels), and trace lines and areas (closed regions filled with a color).
  • erase: to erase items, and lines/areas, or portions of them, from the sketch.
  • edit [A]: for pointwise changes and item properties editing, including item deleting.

Shift and zoom are possible even in draw and in erase mode using two fingers. Zoom controls can be enabled (either temporarily or permanently). If temporary, they show up when you tap at the bottom of the canvas in the middle. If "side-drag" is enabled, the sketch can be shifted by "dragging" it from the left, right and bottom sides of the canvas.

BUTTONS

The modes are represented by the three green buttons on the left: draw, erase, and edit. When one of them is "on" the relative button has a bright green color. Otherwise the mode is "move".

When a green button is "on", a long tap on it, opens the dialog to edit the settings relevant to the mode of the button.

The remaining buttons differ by the mode. In the move mode they are

  • Data Download. The last download shots are bluish to help distinguish them from the others. If no DistoX is selected, this button is hidden.
  • Bluetooth reset/controls
  • Notes
  • Drawing references: legs, splays, stations, metric grid, and scalebar.
  • Plan/Profile switch.
  • "extend" reference

The draw mode has four buttons besides the green ones:

  • Undo removes the last items you have drawn, one at a time.
  • Redo puts the removed item back in the sketch, one at a time. The list of removed items is emptied when you draw something new. Pointwise editing actions may also affect the list of redo.
  • Drawing tool picker
  • Line continuation

The erase mode has three buttons besides the green ones:

  • Undo
  • Redo
  • Filter to limit the eraser to one class among points, lines or areas
  • Size of the eraser [S, M, L]

The edit [A] mode has, besides the green buttons:

  • Filter limits the selectable items [all, points, lines, areas, shots, stations]
  • Size of the selection tool [S, M, L]
  • Point-wise actions
  • Item properties edit
  • Delete the item from the sketch (it asks for confirmation, and can be "undone").
  • Range selection: either pointwise or local (only for lines and areas).

When there are selected items the first two buttons become previous/next navigation buttons in the set of the selected items.

The buttons are somewhat different for x-section sketches. In the move mode the buttons "download", "bluetooth", "switch", and azimuth", are not shown, but there is a button to select which splay to show. It cycles thru four states: "both", "viewed station", "behind station", "none". Accordingly, the drawing references dialog has no box for the splays.

MENUS

  • Export: the sketch can be exported as Therion 2D file(s), cSurvey (a single file), PNG (two files), DXF (two files), and SVG (two files).
  • Statistics. For x-sections this is just the section area.
  • Reload the sketch from a backup copy.
  • Zoom-fit: dialog to fit the sketch into the screen, and to select the portrait or landscape orientation
  • Switch to another sketch.
  • Rename/Delete/Split the sketch. Deleted sketches can be recovered ("recover" menu in shot window)
  • drawing tools palette
  • Overview window, showing all the survey sketches together. [Only for plan and extended profiles]
  • Settings
  • Help

SKETCH SETTINGS

  • Drawing tool picker: most-recent [default], or list, or grid, or triple-grid view
  • Number of recent tools to display in the recent tool picker [3, 4, 5, or 6]
  • Side drag: enable panning by side-drag
  • Zoom controls: specifies whether and how zoom controls are displayed on multitouch devices [B]. The zoom controls can be hidden, displayed only for a short interval when the user tap the lower part of the screen in the middle, or shown all the time
  • Midline check: check if all the legs are attached to the midline
  • xtherion areas save sketch th2 files with an extra point in area borders
  • POINT TOOLS
    • Non-zoomed point: point items stay of fixed size when zooming
    • Base scale for point symbols
    • Label text size [24 pt]
  • LINE TOOLS
    • Line tracing style: fine, normal, coarse, or "splines" (interpolated-curve)
    • Line thickness
    • Direction tick length for x-section lines [5 px]
    • Line point spacing: distance between line points in normal style [5 px]
    • Bezier accuracy determines the number of base points in the interpolating spline. The smaller the more points [1.0]
    • Bezier corner. Even a spline should have corners, when there is a sharp turn at a point. Small values (0.2) make the corner sharper. Large values (20.0) make them rounded.
  • SKETCH ITEMS
    • Splay plan-view threshold: splay shots with inclination above this threshold are hidden in the plan-view [80 deg.]
    • Splay dash coherence: splay shots dashed coherently between plan and profile views [yes]
    • Splay dash clino threshold: splay shots with inclination above this threshold are dashed, in plan view [50 deg.]
    • Splay dash azimuth threshold: splay shots with azimuth off more than this threshold are dashed, in profile view [60 deg.]
    • Splay dash angle threshold: splay shots making an angle smaller than this threshold are dashed, in x-sections [60 deg.]
    • X-section H-threshold: the inclination angle above which a x-section is horizontal. (Below it x-section are vertical).
    • Survey line thickness (midline/splay)
    • Station name text size [18 pt] [N]
    • Size of green dots displayed in edit mode [5 px] [A]
    • Selection radius for fine-editing a sketch; [16 px] [A]
    • Radius of pointing: to select you must not move the finger more than this [A]
    • Eraser size
    • Shift sensitivity: maximum stepping when moving a sketch [60 px] [E]. Increase it if sketch shifting is jaggy
  • AUTO WALLS [E] settings

Sketch cache

Once opened the sketches are kept in memory, in a cache. This speeds up switching between the shot-list window and the sketch window, as the sketch do not need to be read from file. The sketch cache is cleared when the survey is closed.

Sketch file format

Each sketch is saved to file whenever it is modified, or when the sketch window changes to a different view. For performance TopoDroid saves the sketches in a binary format.

Point selection [A]

To do some editing you must select a point: either a point item, a line/area point, a station, or a shot midpoint.
To this aim you tap near the point you want. However if there are several points close where you tap, they are all selected, but only one of them is actively being edited. You change the "active" point among the selected set with the Previous and Next buttons.

Line style

TopoDroid has four line styles. With the normal style [default] the lines are drawn as polylines. With the fine style the polyline segments are shorter, while with the coarse style they are longer. In the fourth style lines are interpolated replacing the segments with cubic Bezier curves (splines). These segments pass through the two endpoints and have two control points used define the curvature. If you export the drawing in DXF with AutoCAD version 6 do not use splines because they get exported as polylines.

Line continuation

The Continuation button is a multi-state button.
  • off: the lines of the current type cannot be joined
  • no: the current type line will not be joined
  • start: the current type line will be joined to a line of the same group if its start-point is close to one of that line endpoints
  • end: the current type line will be joined to a line of the same group by its end-point
  • both: the current type line will be joined to a line of the same group by both its start and its end-point
  • continue: the current type line will continue a line of the same group if its start-point is close enough to an endpoint of that line

Portrait/Landscape presentation

In portrait presentation the North and the Vertical are upward in plan and profile view, respectively. In landscape presentation they are leftward so that the East is upward in plan view and the Right is upward in profile view.
In landscape presentation the iconic symbols are rotated so that they appear natural holding the device in portrait mode.
The sketches are always saved and exported with the North and the vertical upward.

Warning. If the sketch disappear from the display, switch between plan and profile to force TopoDroid to refresh it.
